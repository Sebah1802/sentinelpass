About SentinelPass

What is SentinelPass?

SentinelPass is a cybersecurity-focused password analysis platform designed to help users evaluate password strength, detect breached passwords, and generate secure passwords.


Key Features

  • Password Strength Analysis
  • Security Score Calculation
  • Entropy Measurement
  • Password Crack Time Estimation
  • Secure Password Generator
  • Real Breach Detection using Have I Been Pwned (HIBP)
  • PDF Security Report Generation
  • Custom Themes (Dark, Light, Blue,Green, Purple)

Why SentinelPass?

Weak and reused passwords remain one of the leading causes of cybersecurity incidents. SentinelPass helps users identify weak passwords, understand security risks, and create stronger credentials to improve online safety.


Developed By

Developed by Sebah Mariyam K M as a cybersecurity project built using Python, Flask, HTML, CSS, and JavaScript. SentinelPass focuses on password security analysis, breach detection using Have I Been Pwned (HIBP), secure password generation, and cybersecurity awareness.
